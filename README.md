# learn_5 -- just learning

```
$ cd <FOLDER_NAME>

# compile all in folder
$ make all

# OR compile only C program
$ make c1

# OR compile only Nim program
$ make c2

# run C program
$ make run1

# run Nim program
$ make run2

# run poop benchmark C and Nim programs
$ make poop12

# delete C compiled
$ make clean1

# delete Nim compiled
$ make clean2

# delete all compiled
$ make clean
```
